# N-Tesla-Investigation

FBI Tesla archival investigation project. FOIA-based, evidence-driven, structured database approach.

## Quick Reference
- **Language:** Python 3.11, managed by uv
- **Dependencies:** pdf2image, pytesseract
- **Data:** JSON (relational) + Markdown (analysis prose)
- **Node IDs:** ENT-xxx, RED-xxx, LED-xxx, PAT-xxx, EVT-xxx, LOG-xxx, Q-xxx
